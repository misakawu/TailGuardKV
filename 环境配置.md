## 8. 数据集资产

Pilot 请求数据位于：

```text
/DATACENTER3/zhenxiang.wang/resource/tailguardkv_pilot/requests/longbench_xsum_pilot.jsonl
/DATACENTER3/zhenxiang.wang/resource/tailguardkv_pilot/manifest.json
```

生成命令：

```bash
conda run -n tailguardkv-base python scripts/prepare_pilot_assets.py --download-data --hf-endpoint https://hf-mirror.com
```

Pilot 数据按论文规划的最小规模准备：1 个 QA/长上下文任务加 1 个摘要任务，每类 200-300 请求。本轮选择 LongBench `qasper` 200 条和 XSum validation 300 条。数据生成时先按 `qasper`、`multifieldqa_en`、`hotpotqa` 顺序选择首个可稳定下载且满足下限的 LongBench QA 类配置。每类按原始 split 顺序取前 300 条有效样本；不 shuffle、不重采样、不去重、不改写原始问题、上下文、文档或参考答案。每类前 50% 标记为 `calibration`，后 50% 标记为 `eval`。当前机器无法直连 `huggingface.co`，资产下载使用 `https://hf-mirror.com`。

已有与第三方源码绑定的小型样例数据包括：

| 来源 | 路径 | 说明 |
| --- | --- | --- |
| KIVI | `third_party/KIVI/passkey_examples.jsonl` | KIVI passkey 示例 |
| KIVI | `third_party/KIVI/config/*.json` | LongBench/model/prompt 配置 |
| H2O | `third_party/H2O/h2o_hf/data/xsum.jsonl` | H2O summarization 示例数据 |
| H2O | `third_party/H2O/h2o_hf/data/xsum_opt.jsonl` | H2O OPT 示例数据 |

后续若加入 ShareGPT、LongBench、代码任务或摘要任务数据，应统一放入资源目录或项目外共享目录，并在本文补充绝对路径、版本、样本规模与预处理脚本。
